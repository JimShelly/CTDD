Software Systems Development

Provide a user-friendly web portal and web services for information sharing

Using cloud computing resources, we leverage and customize Atlassian’s Confluence platform to host partner portals for information sharing and collaboration.

 

 

 

Develop Restful state APIs necessary to handle the need of interoperable data exchanged between sites and core system components

We have extensive expertise in the development of transactional and large-scale database architectures to facilitate batch data analytics and real-time machine learning applications. This includes customized data warehouse design, ETL process design & optimization, application integration, server hosting & administration, and development of tailored APIs to support network/web-based access to hosted data.

 

Developed Virtual Biospecimen Inventory System for Clinical Trials

The biospecimen data inventory and tracking system allows real-time access to the core components necessary for clinical trials to the extent possible. We have developed real-time mobile applications for tracking purposes can leveraged and to provide a mobile friendly version of the biospecimen electronic inventory for remote access. Separate identifiers can be used for reference to specimens independent of clinical data. Inventory can be matched with specimen usage consent, clinical annotations and trial data to identify translational research project feasibility. We can also use the Rave® and data exchange functionality already developed to track data on biospecimens collected as part of a trial using standard forms, their shipment and distribution for analysis.

 

Developed systems to facilitate data and specimen sharing

The CTDD systems development team has worked with both NCI funded programs and industry partners to produce innovative systems for the efficient and secure exchange of data and specimens.
